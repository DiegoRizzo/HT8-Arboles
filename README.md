<h3>Propósito del Ejercicio</h3>
<p>El objetivo de este ejercicio es desarrollar una simulación de un sistema operativo utilizando Python. Se busca comparar la eficiencia temporal y la cantidad de iteraciones por medio de la implementación de diferentes Árboles binarios. En este caso, se realizó la implementación de un Binary Search Tree (BST), un Splay Tree y un Red-Black Tree.</p>

<h3>Estructura del Código y Ejecución</h3>
<p>Para llevar a cabo el programa, se hizo uso de cuatro clases, desarrolladas en archivos separados y utilizados como módulos a importar. La clase Main es la clase en la que se realiza la ejecución del programa. Las otras clases son las implementaciones de Árboles y nodos, las cuales definen los procesos de búsqueda e inserción de datos.</p>
